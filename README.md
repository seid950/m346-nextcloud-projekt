## Ziel
Nextcloud Community Edition ohne Docker und ohne Web Installer auf einem Webserver installieren.
Die Datenbank läuft auf einem separaten Server mit MySQL. Beim Aufruf der Server-IP erscheint
der Nextcloud Installationsassistent. Die Verbindungsdaten zur Datenbank werden im Terminal
ausgegeben.

## Struktur (geplant)
- `docs/` – Projekt-Dokumentation (Markdown)
- `scripts/` – Installations- und Hilfsskripte
- `cloud-init/` – (optional) Automatisierung über Cloud-Init
